# 🏏 IPL Performance Analysis using Python, SQL & Power BI

An interactive Data Analytics project that analyzes IPL batting performance using Python for data preprocessing, SQL for data analysis, and Power BI for dashboard visualization.

---

## 📌 Project Overview

This project focuses on transforming raw IPL data into meaningful business insights. The dataset is cleaned and preprocessed using Python, analyzed with SQL, and visualized through an interactive Power BI dashboard.

---

## 🚀 Features

- Data Cleaning using Pandas & NumPy
- Missing Value Handling
- Duplicate Record Removal
- Feature Engineering
- Exploratory Data Analysis (EDA)
- SQL-Based Data Analysis
- Interactive Power BI Dashboard
- KPI Cards & Dynamic Filters

---

## 📊 Dashboard Insights

- Total Runs
- Total Matches
- Strike Rate
- Boundary Analysis (4s & 6s)
- Runs by Season
- Runs vs Opponent
- Dismissal Analysis
- Interactive Season Filter
